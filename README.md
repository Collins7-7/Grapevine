#Grapevine App
This is a blog app where users can have a couple different functionalities within their disposal.
Users can signup into the app, login and get to view all that is in the app.
Users can get to read one blog post by simply clicking on the name of the particular blog post they want to read. 
A user with an account can make a new blog post.

Click [HERE](https://grapevine.onrender.com "Grapevine") for a demo of how the app works.
